QA Checklist

Дизайн

✓ Візуальна відповідність

Макети відповідають бренд-гайдам та дизайн-системі

✓ Типографіка

Правильні шрифти, розміри та міжрядкові інтервали

✓ Кольори та контраст

Доступність кольорів та достатній контраст

Інтерфейс

✓ Responsive дизайн

Коректне відображення на різних пристроях

✓ Інтерактивність

Правильна робота hover, focus та click станів

✓ Навігація

Зрозуміла та логічна структура навігації

Контент

✓ Тексти та заголовки

Без помилок, зрозумілі та відповідні контексту

✓ Зображення та іконки

Якісні зображення з правильними alt-текстами

✓ Форми та валідація

Правильна робота форм та повідомлення про помилки
